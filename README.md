# Colette

**Collaborative LLM Engine for Total Technology Engineering**

Multi-agent AI system that autonomously performs end-to-end software development for web applications.

## Quick Start

```bash
# Install dependencies
make install

# Start dev services (PostgreSQL, Redis, Neo4j)
make docker-up

# Run tests
make test

# Run all quality checks
make check
```
